Raspberry Pi setup

To set up your Raspberry Pi so that it can communicate with the robot arm, you will need to follow a few steps.
  1. Go to the Raspberry Pi setup application and under interfaces, turn on SSH.
  2. Open a new terminal window and run the following: pip3 install HHS-Robotarm It should install without any errors.
